  JALES-BANK
O Jales Bank foi desenvolvido como um estudo de design de alto padrão, inspirado em bancos de elite.  O objetivo foi criar uma interface moderna, elegante e confiável, com forte identidade visual.

 Funcionalidades:

- Cartões 3D flutuantes  
- Design dark com detalhes dourados  
- Seção de notícias fictícias  
- Parcerias com times fictícios  
- Animações suaves (scroll, hover e entrada)  
- Layout responsivo  

---

 Design:

- Preto, dourado, azul escuro e branco  
- Estilo minimalista e sofisticado  
- Tipografia elegante  
- Espaçamento amplo
